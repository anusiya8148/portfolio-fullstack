#  Portfolio Fullstack Website

This is my Fullstack Portfolio Website built using:

- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js, Express.js  
- Database: MongoDB Atlas  
- Deployment: Render  

---

##  Features

- Display projects dynamically from MongoDB
- REST API for project data
- Responsive UI
- Fullstack integration

---

##  Installation

1. Clone the repository
